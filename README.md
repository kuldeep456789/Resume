# ATS Resume Builder & Analyzer

A premium, AI-powered resume building and ATS scoring platform.

## 🚀 Quick Start

### 1. Backend Setup
```bash
cd Backend
npm install
npm start
```
*Create a `.env` file in `Backend/` with your `GROQ_API_KEY`.*

### 2. Frontend Setup
```bash
cd Frontend
npm install
npm run dev
```

---

## 📁 Project Structure

- `Frontend/`: React application (Vite).
- `Backend/`: Node.js Express server for AI analysis.

## 🌟 Key Features
- **AI-Powered ATS Scanning**: Deep resume analysis using Llama 3.3.
- **Location-Based Insights**: Real-time hiring data based on your city.
- **Role Selection**: Targeted analysis for specific tech roles.
- **Secure Architecture**: API keys are protected on the backend.
